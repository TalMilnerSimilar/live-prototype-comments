Step 1: Copy the Widget

Copy comments-widget.js to your project directory.

Step 2: Add the Script Tag

Add this single script tag to your HTML page:

<script defer 
    src="./comments-widget.js" 
    data-endpoint="https://your-netlify-site.netlify.app/.netlify/functions/comments">
</script>

Step 3: Deploy the API

Deploy the Netlify function to your own Netlify account:

  • Copy netlify/functions/comments.mjs
  • Copy package.json and netlify.toml
  • Deploy to Netlify
